# Pharmaceutical_Candidate_for_Breast_Cancer_InSilico_Evaluation_with_Vina_AutodockTools_PyMol-
Computational molecular docking pipeline for evaluating a trimolecular phytopharmaceutical matrix against multiple breast adenocarcinoma targets (HER2/ErbB2, ERα, VEGFR2, and COX-2) using AutoDock Vina, PDB, and PubChem.
